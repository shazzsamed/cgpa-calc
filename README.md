
# CGPA Calculator ACOE (Anna Univeristy)




## Requirements

- Install Google Chrome for login and extracting grades
- Install required dependencies using:

     ```pip install selenium beautifulsoup4```
## How to use

### 1. Run the main.py script

After installing dependencies run the ``main.py`` script. This will prompt a Google Chrome Browser

### 2. Login 

![Screenshot 2025-03-05 194116](https://github.com/user-attachments/assets/7124b3d7-f897-47b5-b96a-a06c8033fa86)

![image](https://github.com/user-attachments/assets/b31e9f36-ef0f-434f-98aa-ce330c66a693)


Login using your acoe credentials aand navigate to the "Attendance and Marks" Section

### 3. Extracting Grades

Once you naviagte to the "Attendance and Marks" section, click ENTER on the terminal were script is running. Now the script will extract grade and calculate GPA and finally the CGPA.

![image](https://github.com/user-attachments/assets/1ebd0802-4b8a-4beb-8533-808a0de97779)

### 4. Final result

The final result will be shown in the terminal something like this.

## Contributing to grade credits

This repository only contains credits of courses for M.Sc(5 YRS Integarted) Computer Science. Other courses credits can be added to the ``credits.py``. Also looking for better options to organize and integarte credits for the courses rather than using the current system.
